# 🚀 Space Shooter 2D Game

A thrilling 2D space arcade shooter game built using the **Godot Engine**. Navigate through dangerous galaxies, dodge obstacles, and destroy enemy alien ships across multiple challenging levels.

---

## 🎮 Game Features

- **Classic 2D Arcade Mechanics:** Smooth spaceship movement and satisfying shooting controls.
- **Progressive Difficulty:** Features 6 unique levels packed with different enemy types.
- **Interactive UI:** Fully functional Main Menu, Level Selection screen, and in-game HUD.
